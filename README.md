# Описание
Sysmon-parser.exe - используя SystemMonitor.exe от Mircosoft собирает данные о процессах и дополняет их сторонними метриками собранными в ходе работы процесса.
Py файлы - предназначенны, чтобы отсортировать собранные данные убрав дубликаты и представив в результате данные в формате .json для каждого отделяемого от другого по ключу GUID процесса. Для визуализации запуска процессов в виде дерева, для анализа. И для дальнейшего формирования датасетов для обучения нейросети.


# Установка и запуск

## Установка Sysmon
1) Разархивировать SYSMON.zip в папку

## Запуск Sysmon драйвера, для начала логирования
1) Открыть powershell (от имени администратора) и перейти в папку из п. 1)
2) Прописать команду: 
``` powershell
.\Sysmon64.exe -i sysmonconfig-research.xml
```

## Выгрузка Sysmon драйвера из системы и остановка его работы (обязательно, после окончания работы)
1) Открыть powershell (от имени администратора) и перейти в папку из п. 1)
2) Прописать команду: 
``` powershell
.\Sysmon64.exe -u
```
# Запуск программы

1) Открыть powershell (от имени администратора) и перейти в папку, где лежит Sysmon-parser.exe
2) Прописать команду:
``` posershell
.\Sysmon-parser.exe
```
Чтобы завершить выполнение программы и сохранить собранные данные просто прекращаем ее выполнение Ctrl+C или закрываем Powershell
